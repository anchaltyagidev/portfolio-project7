Web design refers to the process of conceptualizing, planning, and creating the visual elements and layout of a website. It encompasses the aesthetic aspects of website development, focusing on the overall look,feel,and user experience.Effective web design aims to create a cohesive and intuitive user experience while aligning with the website's purpose and goals.

Database

DBMS stands for Database Management System. It's a software suite that provides an interface for users to interact with a database. It enables the creation, querying, updating, and management of databases, ensuring data integrity, security, and efficient access. Examples include MySQL, Oracle, SQL Server, and MongoDB.
